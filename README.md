# Namma Bazaar Bengaluru

Namma Bazaar Bengaluru is a static e-commerce storefront for a neighborhood store. Customers can browse products, search by keyword, filter by category, add products to a cart, and place a demo pickup order.

## Features

- Product listing cards with generated images, descriptions, and INR pricing
- Search and category filters for quick browsing
- Slide-out cart with quantity updates, remove actions, and live totals
- Pickup fee logic: Rs.40 for orders below Rs.1200, free at or above Rs.1200
- Checkout form with confirmation modal
- Cart persistence using localStorage
- Responsive layout for desktop and mobile

## Run Locally

Open index.html directly in your browser, or start a simple local server:

```powershell
py -m http.server 4173
```
